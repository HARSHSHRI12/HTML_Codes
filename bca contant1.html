History of C:-

C language is a general-purpose and procedural-oriented programming language

that was developed by Dennis Ritchie in 1972 at the AT&T Bell Laboratories.It was

developed as a system implementation language for the emerging Unix operating system to beat

the issues of previous languages such as B and BCPL.23 C evolved a type structure from thetypeless

language BCPL and started on a small machine as a tool to better a minimal programming environment.
